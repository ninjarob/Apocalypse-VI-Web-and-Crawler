# Apocalypse VI MUD - Game Documentation

This documentation covers the core gameplay mechanics, world systems, and features of the Apocalypse VI MUD (Multi-User Dungeon).

## Overview

Apocalypse VI is a text-based multiplayer RPG set in a fantasy world with complex character systems, magic, combat, and social interactions. The game features a rich world with 74 zones, multiple races and classes, extensive magic systems, and persistent player interactions.

New players should begin by reading the [Character Creation](character-creation.md) guide to understand the available races and classes. The [World Geography](world-geography.md) section provides essential navigation information.

## Key Commands

- `look` - Examine your current room
- `scan` - See nearby entities and their locations
- `time` - Check current time and date
- `who -z` - List players in your current zone
- `exits` - Show available room exits

## Game Features

- **Real-time gameplay** with persistent world state
- **Complex character progression** through experience and leveling
- **Magic system** with mana costs and spell casting
- **Social interactions** with other players and NPCs
- **Economic system** with shops and services
- **Zone-based player visibility** and interactions

## Technical Notes

This game runs on a custom MUD engine with:
- Real-time tick system (approximately 60-second intervals)
- Dynamic NPC movement and behavior
- Persistent player statistics and inventory
- Zone-based chat and player listings
